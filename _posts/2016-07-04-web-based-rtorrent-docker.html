web based rtorrent with docker, with this very simple below command
then see it live on http://url:8086
and user password is
  docktorrent/p@ssw0rd

docker run -it \
    -p 8086:80 -p 45566:45566 -p 9527:9527/udp \
    --dns 8.8.8.8 \
    -v /data-store:/rtorrent \
    -e UPLOAD_RATE=1024 \
    kfei/docktorrent
