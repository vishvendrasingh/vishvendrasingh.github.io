add js or css depending on browser with php with the below short code:


<?=    (preg_match('/Safari/i',$_SERVER["HTTP_USER_AGENT"] )) ? 'body{line-height:1.2; margin:0;padding:0;} @media print { html, body { margin: 0; padding: 0; }' : '' ?>
